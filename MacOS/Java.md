# Java

## Install via Brew (One version only)

```
brew tap AdoptOpenJDK/openjdk
brew cask install adoptopenjdk8
```

## Install via SDKMan

### SDKMAN

##### Install via curl

```
curl -s "https://get.sdkman.io" | zsh
// new terminal
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

#### Install java

```
sdk install java 8.0.222-zulu
```
